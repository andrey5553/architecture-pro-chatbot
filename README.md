# Задание 3. Создание векторного индекса базы знаний

  ## Общие сведения
  Использовал модель all-mpnet-base-v2.
  Эта модель считается "тяжеловесом" в семействе Sentence-Transformers и часто рекомендуется, 
  когда качество важнее скорости и размера модели.
  
  Выдающееся качество: all-mpnet-base-v2 стабильно показывает одни из лучших результатов на бенчмарках 
  для задач семантического поиска и определения схожести текстов. 
  Например, её средняя производительность оценивается в 63.3, что значительно выше, 
  чем у более легкой all-MiniLM-L6-v2 (58.8) . 
  
  Это делает её отличным выбором для production-систем, где точность критична.
